# bank.sinopac.com 品牌簡介
永豐銀行（Bank SinoPac）致力於提供全面、創新且友善的金融服務，滿足個人及企業客戶的多樣需求。品牌的願景是成為客戶信賴的金融夥伴，並以數位轉型推動金融服務的便利性與效率。

## 核心價值
- 客戶至上：始終以客戶需求為中心，提供優質服務。
- 創新與數位化：積極推動數位金融創新，提升用戶體驗。
- 誠信與透明：堅持誠信經營，確保業務運作的透明性。
- 社會責任：關注環境、社會及公司治理（ESG），致力於永續發展。

## 主要服務

| 服務名稱         | 服務簡介                                                         |
| ---------------- | ---------------------------------------------------------------- |
| 個人化通知服務   | 提供客戶即時的個人化金融通知，提升用戶互動體驗。               |
| 智能理財         | 利用智能演算法提供個性化的投資建議，幫助客戶管理資產。         |
| 信用卡服務       | 提供多樣化的信用卡產品，並附帶各種優惠及回饋計劃。             |
| 企業金融服務     | 為企業客戶提供現金管理、融資及自動化金流工具。                 |
| 投資組合分析     | 提供客戶個性化的投資組合建議，協助資產配置與風險管理。         |

## 產品特色
- 獨特的信用卡優惠計劃，提供高回饋率和專屬優惠。
- 數位帳戶及智能理財服務，簡化開戶和資產管理流程。
- 個人化的客戶服務，透過智慧小豐提供即時的客戶支援。
- 多幣別存款與外匯服務，滿足客戶的國際金融需求。

## 常見問答（FAQ）

**Q：如何開設永豐銀行的數位帳戶？**  
A：客戶可透過永豐銀行的官方網站或行動APP進行數位帳戶的線上申請，簡單方便。

**Q：永豐銀行的信用卡有哪些優惠？**  
A：永豐銀行提供多種信用卡，包含現金回饋、分期付款及特定商戶的專屬優惠。

**Q：如何使用智能理財服務？**  
A：客戶只需設定投資目標，系統將自動為您提供投資建議，並定期調整組合。

**Q：永豐銀行對於ESG的承諾是什麼？**  
A：永豐銀行致力於推動社會責任，專注於環境保護、社會貢獻及良好的公司治理。

**Q：如何查詢我的信用卡回饋？**  
A：客戶可透過永豐銀行的行動APP或官方網站查詢信用卡的回饋及消費明細。
