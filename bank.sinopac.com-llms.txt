# bank.sinopac.com 品牌簡介
永豐銀行致力於提供全面的金融服務，透過數位化轉型，讓客戶享受便捷、安全的銀行體驗。品牌的願景是成為客戶最信賴的金融夥伴，提供創新及個性化的金融解決方案，滿足不同客戶的需求。

## 核心價值
- 客戶為先：始終以客戶需求為導向，提供優質服務。
- 創新：不斷推動數位化轉型，提供便捷的金融服務。
- 負責任：承擔社會責任，推動永續金融與環保行動。
- 專業：擁有專業的團隊，提供全面的金融知識與建議。

## 主要服務

| 服務名稱         | 服務簡介                                                         |
| ---------------- | ---------------------------------------------------------------- |
| 數位帳戶 (DAWHO) | 提供便捷的數位化銀行服務，無需實體開戶即可進行存款、轉帳及投資。 |
| 信貸服務         | 提供個人及企業的信用貸款，簡化申請流程，快速撥款。               |
| 美股投資         | 透過DAWHO數位帳戶，輕鬆投資美股，無需匯款或開證券戶。             |
| 小蜜豐點數計畫   | 提供客戶參與數位行銷活動以獲得金Bee點數，並可用於兌換商品。      |
| 信託服務         | 提供專業的資產管理及信託服務，協助客戶規劃未來財務需求。         |

## 產品特色
- 數位化服務：無需到行，隨時隨地進行銀行交易。
- 高利率存款：DAWHO數位帳戶提供高達1.5%的活儲利率。
- 自動化投資顧問：透過永豐ibrAin提供智能化的投資建議。
- 多元化的金融產品：涵蓋存款、貸款、保險及投資等多種服務。

## 常見問答（FAQ）

**Q：如何開設DAWHO數位帳戶？**  
A：您可以透過永豐銀行的網站或APP進行線上申請，提交必要的身份證明文件即可。

**Q：DAWHO帳戶的轉帳限額是多少？**  
A：轉帳限額根據客戶的認證方式而異，詳情可參考官方說明。

**Q：小蜜豐點數如何獲得？**  
A：參加永豐銀行的數位行銷活動或完成指定任務即可獲得金Bee點數。

**Q：如何申請信用貸款？**  
A：申請人需年滿18歲，並提供身份證明及收入證明可進行線上申請。

**Q：DAWHO帳戶的投資功能有什麼？**  
A：客戶可以透過DAWHO帳戶進行美股及基金的投資，並享有豐富的市場資訊。
