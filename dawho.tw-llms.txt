# dawho.tw 品牌簡介
DAWHO是一個以數位金融服務為核心的品牌，致力於提供用戶全方位的金融解決方案，包括開戶、投資、貸款及理財服務。品牌的願景是成為數位金融領域的領導者，透過創新的科技和貼心的服務，讓每位用戶都能輕鬆掌握自己的財務狀況，實現財務自由。

## 核心價值
- **創新**：不斷引入新技術，提升用戶體驗。
- **透明**：提供清晰的金融資訊，讓用戶做出明智的決策。
- **責任**：承擔社會責任，推動金融教育及防詐宣導。
- **客戶至上**：以用戶需求為導向，提供個性化的服務。

## 主要服務

| 服務名稱         | 服務簡介                                                         |
| ---------------- | ---------------------------------------------------------------- |
| 數位帳戶         | 提供高利息活儲、靈活投資及即時交易功能，簡化開戶流程。         |
| 信貸服務         | 提供快速撥款的信貸方案，年利率低至0.01%，滿足用戶的資金需求。 |
| 房貸服務         | 提供理財型房貸，資金隨借隨還，按日計息，靈活運用資金。       |
| 信用卡           | 現金回饋信用卡，提供多種消費回饋方案，提升用戶的消費價值。   |
| 投資服務         | 提供基金定期定額交易、股票交易及智能理財顧問服務。           |

## 產品特色
- **高利活儲**：DAWHO數位帳戶年利率最高可達1.5%。
- **靈活資金運用**：理財型房貸允許隨借隨還，無手續費。
- **多幣卡回饋**：外幣消費享最高2.5%回饋，無上限。
- **智能理財**：ibrAin提供個性化的投資組合建議，降低投資風險。

## 常見問答（FAQ）

**Q：DAWHO數位帳戶的開戶流程是什麼？**  
A：用戶可以透過DAWHO官方網站或APP進行線上開戶，需提供身份證明及其他必要文件。

**Q：如何申請DAWHO輕房貸？**  
A：用戶可於DAWHO APP內填寫相關資料，並提交申請，專人將會聯繫協助。

**Q：DAWHO信用卡的回饋政策是什麼？**  
A：持卡人在指定通路消費可享最高12%現金回饋，並有其他日常消費回饋。

**Q：如何防範網路詐騙？**  
A：永豐銀行提供防詐教育，建議用戶不點擊不明連結，並定期檢查帳戶安全。

**Q：DAWHO提供的投資服務有哪些？**  
A：DAWHO提供台股、美股的交易服務，並有基金定期定額的手續費優惠。
