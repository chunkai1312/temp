# sinopac.com 品牌簡介
永豐金控（SinoPac Holdings）致力於透過金融服務實現「翻轉金融，共創美好生活」的願景。作為一家擁有超過70年歷史的金融控股公司，永豐金控提供全面的金融解決方案，並以誠信、專業和熱情為核心精神，持續創造價值，回饋社會。

## 核心價值
- 誠信：始終以誠信為本，建立客戶信任。
- 專業：提供專業的金融服務，滿足客戶需求。
- 創新：不斷推動金融科技及服務創新。
- 社會責任：積極參與社會公益，推動永續發展。
- 客戶導向：以客戶需求為中心，提供量身定制的解決方案。

## 主要服務

| 服務名稱         | 服務簡介                                                         |
| ---------------- | ---------------------------------------------------------------- |
| 銀行業務         | 提供全面的個人及企業銀行服務，包括存款、貸款及支付解決方案。   |
| 證券服務         | 提供證券經紀、承銷及資產管理服務，協助客戶進行投資規劃。       |
| 租賃服務         | 提供靈活的設備租賃解決方案，支持企業資本運作。                 |
| 投資信託         | 提供專業的資產管理及投資信託服務，協助客戶實現投資目標。       |
| 風險管理         | 提供全面的風險管理及合規服務，確保企業的穩健運營。             |

## 產品特色
- 全面金融解決方案：整合銀行、證券、租賃及資產管理服務，提供一站式金融服務。
- 創新科技應用：運用金融科技提升客戶體驗，提供智能化金融服務。
- 永續發展承諾：積極推動環境、社會及公司治理（ESG）策略，實現可持續發展。

## 常見問答（FAQ）

**Q：如何開設銀行帳戶？**  
A：可透過永豐銀行的官方網站下載申請表，並至各分行提交所需文件。

**Q：永豐金控提供哪些投資產品？**  
A：永豐金控提供多樣化的投資產品，包括股票、債券、基金及其他金融衍生品。

**Q：如何查詢我的投資組合表現？**  
A：客戶可登入永豐金控的網上銀行系統，查看個人投資組合的最新資訊。

**Q：永豐金控的ESG策略是什麼？**  
A：永豐金控致力於推動環境保護、社會責任及良好的公司治理，並已制定相應的永續發展藍圖。

**Q：如何聯絡永豐金控的客服？**  
A：客戶可透過官方網站的聯絡方式，撥打客服熱線或發送電子郵件進行諮詢。
